# Phishing Attack Analysis

**This repository contains documentation of a simulated phishing attack in a controlled lab environment. The goal was to gain deeper understanding of offensive tools like Storm-Breaker and NGROK.**

[Click here to view the documentation](phishingAnalysis.md)

**Tools used:**
1. VirtualBox to create a safe environment.
2. Storm-Breaker for data capturing.
3. NGROK for network tunnel making.

**Skills gained/used:**
* Security Analysis (Identifying IOCs)
* Lab Configuration (Creating & configuring VM)
* Network Awareness (Understanding unencrypted data flow and port usage)
